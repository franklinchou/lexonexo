About me

Automate all the things!
-- Franklin Chou (age: 10)

I'm Franklin Chou. I'm a 3L at Seton Hall Law School. I started developing software in my late teens to automate boring stuff. The human mind is immensely powerful. But it gets bored easily and when it gets bored it makes mistakes. So I thought: Why not leave the boring stuff to machines? Many years later, I find that there is still a lot of stuff that needs automation. {{ config.PROJ_NAME }} is just one attempt to solve the automation problem.

Colophon

{{ config.PROJ_NAME }} is built on the Flask web application microframework. {{ config.PROJ_NAME }} talks to databases using SQLAlchemy. And database storage is provided by postgreSQL. Asynchronous job queueing is provided by Celery and Celerybeat.

The heart of {{ config.PROJ_NAME }} is in the Selenium web driver automation package. Selenium does all the heavy lifting. From verifying Lexis logins to performing legal search queries, {{ config.PROJ_NAME }} ❤s Selenium!
